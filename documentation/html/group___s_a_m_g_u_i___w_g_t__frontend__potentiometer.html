Detailed Description

The Potentiometer driver posts messages to the Widget message queue.

Board pontentiometer is connected to the 12 bit ADC analog line. Each time the user turns the potentiometer, it triggers an interrupt which posts a message with a value from 0 to 255 corresponding to the measurement.

The SAMPLING_PERIOD definition controls the sampling frequency. A timer counter in waveform mode is used to trigger acquisitions. It is configured to count from the Slow clock frequency.

The window mode of the ADC is used to trigger an ADC interrupt only when the measure goes outside a window controlled by the NB_INTERVALS definition.

Each time a new ADC interrupt occurs, a new window is calculated from the current measurement.

During the initialization a first measure is done to get the initial value. Application can have access to the current measure using ioctl command.

For more accurate information, please look at the ADC section of the Datasheet.
