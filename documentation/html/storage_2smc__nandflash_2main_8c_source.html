00030 /**
00031  * \page smc_nandflash SMC NAND Flash Example
00032  *
00033  * \section Purpose
00034  *
00035  * This basic nandflash example shall show how to read and write data from/to
00036  * a nandflash connected to the SMC, taking ECC and Bad Block marking into account
00037  *
00038  * \section Requirements
00039  *
00040  * This package can be used with SAM3S evaluation kits.
00041  *
00042  * \section Description
00043  *
00044  * The required steps are:
00045  * <ul>
00046  * <li> Configure the SMC to interface with the NAND Flash. </li>
00047  * <li> Read a page.</li>
00048  * <li> Calculate the ECC by software and check it is correct.</li>
00049  * <li> Prepare a buffer and calculate the ECC by software.</li>
00050  * <li> Write the buffer into a NAND flash page and store the ECC.</li>
00051  * <li> Read the page and check that ECC is correct.</li>
00052  * </ul>
00053  * \section Usage
00054  *
00055  * -# Build the program and download it inside the evaluation board. Please
00056  *    refer to the
00057  *    <a href="http://www.atmel.com/dyn/resources/prod_documents/doc6224.pdf">
00058  *    SAM-BA User Guide</a>, the
00059  *    <a href="http://www.atmel.com/dyn/resources/prod_documents/doc6310.pdf">
00060  *    GNU-Based Software Development</a> application note or to the
00061  *    <a href="ftp://ftp.iar.se/WWWfiles/arm/Guides/EWARM_UserGuide.ENU.pdf">
00062  *    IAR EWARM User Guide</a>, depending on your chosen solution.
00063  * -# On the computer, open and configure a terminal application
00064  *    (e.g. HyperTerminal on Microsoft Windows) with these settings:
00065  *   - 115200 bauds
00066  *   - 8 bits of data
00067  *   - No parity
00068  *   - 1 stop bit
00069  *   - No flow control
00070  * -# Start the application.
00071  * -# In the terminal window, the following text should appear:
00072  *     \code
00073  *     -- SMC NandFlash Example xxx --
00074  *     -- xxxxxx-xx
00075  *     -- Compiled: xxx xx xxxx xx:xx:xx --
00076  *     Nandflash ID is 0x1580DA2C
00077  *     -I-     Nandflash driver initialized
00078  *     -I- Size of the whole device in bytes : 0x10000000
00079  *     -I- Size in bytes of one single block of a device : 0x20000
00080  *     -I- Number of blocks in the entire device : 0x800
00081  *     -I- Size of the data area of a page in bytes : 0x800
00082  *     -I- Number of pages in the entire device : 0x40
00083  *    \endcode
00084  *
00085  * \section References
