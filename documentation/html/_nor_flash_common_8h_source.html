00033  * \section Purpose
00034  *
00035  * The Hardware Adaptation Layer manages the hardware functions of the %NORFLASH
00036  * %Flash. It implements a low level driver to manage the hardware functionality
00037  * of %NORFLASH.
00038  *
00039  * \section Usage
00040  *
00041  * -# WriteRawData() is used to write data to the %NorFlash device.
00042  * -# ReadRawData() is used to read data from the %NorFlash device.
00043  * -# The specified address for read/write opertion is an linear byte
00044  *    address of targer application byte address space represented by
00045  *    WORD((8-bit, 16-bit, 32-bit). The start (base) address of the
00046  *    flash memory in this address space is defined in board.h file
00047  *    as BOARD_NORFLASH_ADDR and users might need to change it.
00048  * -# WriteCommand() is used to write a command to specified command
00049  *    address.
00050  *
00051  * \section NorFlash_Error_Codes
00052  *
00053  * It provides detailed error codes to describe the various errors
00054  * that may occur during the operation. Some functions return an error
00055  * number directly as the function value. These functions return a value
00056  * of zero (NorCommon_ERROR_NONE) to indicate a success.
00057  */
