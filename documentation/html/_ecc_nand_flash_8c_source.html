00030 /**
00031  * \file
00032  *
00033  * EccNandFlash layer is called by SkipBlockNandFlash driver, it will call the bl driver (RawNandFlash)
00034  * to do write/read operations, and do ECC check to the write/read result, it then will feedback the
00035  * ecc check result to the upper SkipBlockNandFlash layer driver.
00036  *
00037  * -# EccNandFlash_Initialize is used to initializes an EccNandFlash instance.
00038  * -# EccNandFlash_WritePage is used to write a Nandflash page with ecc result, the function
00039  *      will calculate ecc for the data that is going to be written, and write data and spare(with
00040  *      calculated ecc) to Nandflash device.
00041  * -# EccNandFlash_ReadPage is uese to read a Nandflash page with ecc check, the function
00042  *      will read out data and spare first, then it calculates ecc with data and then compare with
00043  *      the readout ecc, and feedback the ecc check result to dl driver.
00044  */
