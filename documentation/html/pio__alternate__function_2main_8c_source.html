00031 /**
00032  *  \page pio_alternate_function PIO Alternate Function Example
00033  *
00034  *  \section Purpose
00035  *
00036  *  The PIO Alternate Function demonstrates how to alternate ERASE pin between system
00037  *  I/O mode and PIO mode.
00038  *
00039  *  \section Requirements
00040  *
00041  *  This package can only be used with sam3s-ek.
00042  *
00043  *  \section Description
00044  *
00045  *  The application shows the alternate function of ERASE pin, which extends PIO
00046  *  from a dedicated system I/O line. It first programs the internal flash with
00047  *  a walking pattern and then enable ERASE pin in PIO mode. After the erase
00048  *  operation, it check the content of flash, which is unchanged due to ERASE in
00049  *  PIO mode. In the next step, it put ERASE pin in system I/O mode,which will
00050  *  enable the erase function. The content of flash should be erased to 0xFF, which
00051  *  indicates the ERASE pin functioning.
00052  *
00053  *  The pins which are used by USB(DDM,DDP),JTAG(TCK,TMS,TDO,TDI) and Oscillator(XIN,
00054  *  XOUT,XIN32,XOUT32) could also be alternated between general purpose I/O and system
00055  *  I/O in the similar way.
00056  *
00057  *  \section Usage
00058  *
00059  *  -# Build the program and download it inside the evaluation board. Please
00060  *     refer to the
00061  *     <a href="http://www.atmel.com/dyn/resources/prod_documents/doc6224.pdf">
00062  *     SAM-BA User Guide</a>, the
00063  *     <a href="http://www.atmel.com/dyn/resources/prod_documents/doc6310.pdf">
00064  *     GNU-Based Software Development</a>
00065  *     application note or to the
00066  *     <a href="ftp://ftp.iar.se/WWWfiles/arm/Guides/EWARM_UserGuide.ENU.pdf">
00067  *     IAR EWARM User Guide</a>,
00068  *     depending on your chosen solution.
00069  *  -# On the computer, open and configure a terminal application
00070  *     (e.g. HyperTerminal on Microsoft Windows) with these settings:
00071  *    - 115200 bauds
00072  *    - 8 bits of data
00073  *    - No parity
00074  *    - 1 stop bit
00075  *    - No flow control
00076  *  -# Start application.
00077  *  -# In the terminal window, the
00078  *     following text should appear (values depend on the board and chip used):
00079  *     \code
00080  *      -- PIO Alternate Function Example xxx --
00081  *      -- xxxxxx-xx
00082  *      -- Compiled: xxx xx xxxx xx:xx:xx --
00083  *     \endcode
00084  *  -# Perform erase operation following the application prompt.
00085  *
00086  *  \section References
00087  *  - pio_alternate_function/main.c
