00030 /**
00031  * \page pwm_pdc PWM with PDC Example
00032  *
00033  * \section Purpose
00034  *
00035  * This example demonstrates a simple configuration of three PWM channels to
00036  * generate variable duty cycle signals. The update of the duty cycle values
00037  * is made automatically by the Peripheral DMA Controller (PDC).
00038  * This will cause three LEDs on the evaluation kit to glow repeatedly.
00039  *
00040  * \section Requirements
00041  *
00042  * This package can be used with SAM3S evaluation kits.
00043  *
00044  * \section Description
00045  *
00046  * Three PWM channels (channel #0, #1 and #2) are configured to generate
00047  * a 50Hz PWM signal. The update of the duty cycle values is made
00048  * automatically by the PDC.
00049  *
00050  * When launched, this program displays a menu on the terminal, enabling the user
00051  * to choose between several options:
00052  *   - Set update period for syncronous channel
00053  *   - Set dead time
00054  *   - Set output override
00055  *
00056  * \section Usage
00057  *
00058  * -# Build the program and download it inside the evaluation board. Please refer to the
00059  * <a href="http://www.atmel.com/dyn/resources/prod_documents/doc6132.pdf">
00060  * SAM-BA User Guide</a>, the
00061  * <a href="http://www.atmel.com/dyn/resources/prod_documents/doc6132.pdf">
00062  * GNU-Based Software Development</a> application note or to the
00063  * <a href="http://www.atmel.com/dyn/resources/prod_documents/doc6132.pdf">
00064  * IAR EWARM User Guide</a>, depending on your chosen solution.
00065  * -# Optionally, on the computer, open and configure a terminal application
00066  *    (e.g. HyperTerminal on Microsoft Windows) with these settings:
00067  *   - 115200 bauds
00068  *   - 8 bits of data
00069  *   - No parity
00070  *   - 1 stop bit
00071  *   - No flow control
00072  * -# Start the application.
00073  * -# Depending on the board being used, two LEDs will start glowing repeatedly.
00074  * -# Select one or more options to set the configuration of PWM channel.
00075  *
00076  * \section References
00077  * - pwm_pdc/main.c
00078  * - pwmc.c
00079  * - pwmc.h
