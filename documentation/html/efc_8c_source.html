00030 /** \addtogroup efc_module Working with EEFC
00031  * The EEFC driver provides the interface to configure and use the EEFC
00032  * peripheral.
00033  *
00034  * The user needs to set the number of wait states depending on the frequency used.\n
00035  * Configure number of cycles for flash read/write operations in the FWS field of EEFC_FMR.
00036  *
00037  * It offers a function to send flash command to EEFC and waits for the
00038  * flash to be ready.
00039  *
00040  * To send flash command, the user could do in either of following way:
00041  * <ul>
00042  * <li>Write a correct key, command and argument in EEFC_FCR. </li>
00043  * <li>Or, Use IAP (In Application Programming) function which is executed from
00044  * ROM directly, this allows flash programming to be done by code running in flash.</li>
00045  * <li>Once the command is achieved, it can be detected even by polling EEFC_FSR or interrupt.
00046  * </ul>
00047  *
00048  * The command argument could be a page number,GPNVM number or nothing, it depends on
00049  * the command itself. Some useful functions in this driver could help user tranlate physical
00050  * flash address into a page number and vice verse.
00051  *
00052  * For more accurate information, please look at the EEFC section of the
00053  * Datasheet.
00054  *
00055  * Related files :\n
00056  * \ref efc.c\n
00057  * \ref efc.h.\n
00058 */
