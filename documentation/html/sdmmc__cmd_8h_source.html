00032 /**
00033  * \page sdmmc_cmd SD/MMC Low Level Commands
00034  *
00035  * \section Purpose
00036  *
00037  * Implementation for SD/MMC low level driver. Supply a set of SD/MMC mode's
00038  * interface.
00039  *
00040  * \section sdmmc_prop_setup SD/MMC HW property and setup functions
00041  *
00042  * - SdmmcGetProperty()
00043  * - SdmmcSetBusWidth()
00044  * - SdmmcEnableHsMode()
00045  * - SdmmcSetSpeed()
00046  *
00047  * \section sdmmc_cmd_fun SD/MMC command functions
00048  *
00049  * - Functions for both SD/MMC
00050  *   - SdmmcPowerOn() : Perform SD/MMC power on initialization sequence.
00051  *   - SdmmcCmd0() : Resets all cards to idle state
00052  *   - SdmmcCmd2() : Asks any card to send the CID numbers on the CMD line
00053  *   - SdmmcCmd7() : Command toggles a card between the stand-by and transfer
00054  *                   states or between
00055  *   - SdmmcCmd9() : Addressed card sends its card-specific data (CSD) on the CMD
00056  *                   line
00057  *   - SdmmcCmd12() : Forces the card to stop transmission
00058  *   - SdmmcCmd13() : Addressed card sends its status register
00059  *   - SdmmcCmd16() : Set block length
00060  *   - SdmmcCmd17() : Read single block
00061  *   - SdmmcCmd18() : Read multiple blocks
00062  *   - SdmmcCmd24() : Write single block
00063  *   - SdmmcCmd25() : Write multiple blocks
00064  *   - SdmmcCmd55() : App command, should be sent before application specific
00065  *                    command
00066  *   - SdmmcRead() : Write data without any command
00067  *                   (usually follow SdmmcCmd18()).
00068  *   - SdmmcWrite() : Write data without any command
00069  *                    (usually follow SdmmcCmd25()).
00070  * - Functions for SD card.
00071  *   - SdCmd3() : Ask the card to publish a new relative address
00072  *   - SdCmd6() : SD Switch command.
00073  *   - SdCmd8() : Sends SD Memory Card interface condition, which includes host supply voltage
00074  *                information and asks the card whether card supports voltage
00075  *   - SdAcmd6() : Defines the data bus width
00076  *   - SdAcmd41() : Asks to all cards to send their operations conditions.
00077  *   - SdAcmd51() : Sends SD Card Configuration Register (SCR).
00078  * - Functions for MMC card
00079  *   - MmcCmd1() : MMC send operation condition command.
00080  *   - MmcCmd3() : Set a new relative address to MMC card.
00081  *   - MmcCmd6() : MMC Switch.
00082  *   - MmcCmd8() : Sends MMC EXT_CSD.
00083  */
