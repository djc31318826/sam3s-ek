SD/MMC Driver Interface

Purpose

Implementation for sdcard sd mode physical layer driver. Supply a set of sdcard sd mode's interface.

Usage

  • General Card Support
    1. SD_Init(): Run the SDcard initialization sequence
    2. SD_GetCardType() : Return SD/MMC reported card type.
  • SD/MMC Memory Card Operations
    1. SD_ReadBlock() : Read one block of data
    2. SD_WriteBlock() : Write one block of data
    3. SD_ReadBlocks() : Read blocks of data
    4. SD_WriteBlocks() : Write blocks of data
    5. SD_Read() : Read blocks of data with multi-access command (Optimized read, see SD/MMC Read).
    6. SD_Write() : Read blocks of data with multi-access command (Optimized write, see SD/MMC Write).
    7. SD_GetNumberBlocks() : Return SD/MMC card reported number of blocks.
    8. SD_GetBlockSize() : Return SD/MMC card reported block size.
    9. SD_GetTotalSizeKB() : Return size of SD/MMC card in KBytes.
  • SDIO Card Operations
    1. SDIO_ReadDirect() : Read bytes from registers.
    2. SDIO_WriteDirect() : Write one byte to register.
    3. SDIO_ReadBytes() : Read 1 ~ 512 bytes from card.
    4. SDIO_WriteBytes() : Write 1 ~ 512 bytes to card.
